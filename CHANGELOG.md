# Changelog

## [0.1.1] - 2025-12-12

### Added

- `share` command to share files/folders (--anyone or --email)
- `unshare` command to remove permissions
- `permissions` command to list permissions on a file

## [0.1.0] - 2025-12-12

### Added

- Initial release
- Account management (credentials, add, remove, list)
- List files (ls), search, get metadata
- Download files (with Google Docs export)
- Upload files
- Create folders (mkdir)
- Delete, move, rename files
- URL generation
